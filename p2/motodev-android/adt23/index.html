P2 Update site for the Motodev Core plugins. Items included are:

  • Android Localization Editor
  • Snippets for common Android code. Available in Snippets View
  • MOTODEV Database Explorer
  • File Explorer for browsing files on a running emulator or device
  • View for Managing Signing Keys

These plugins have been tested with Eclipse 3.8, and Eclipse 4.2. You need Sequoyah, and MAT available to be installed.

You need to use Eclipse's INSTALL NEW SOFTWARE option to install. Use this url for the software update site.
